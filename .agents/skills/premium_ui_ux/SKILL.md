# ✨ Premium UI/UX Skill

Skill ini adalah kompas visual untuk memastikan JUSTICE CORE terlihat seperti produk kelas dunia.

## 🎯 Fokus Utama
1.  **Visual Excellence**: Gunakan palet warna HSL curated, dark mode yang elegan, dan tipografi modern (Inter/Inter Tight).
2.  **WOW Effect**: Implementasikan Glassmorphism, Micro-animations, dan Haptic Feedback (untuk mobile).
3.  **Responsive Layout**: Desain harus terasa "native" baik di smartphone layar kecil maupun desktop lebar.

## 🛠️ Aturan Implementasi
- Gunakan Tailwind CSS dan Framer Motion sebagai stack utama styling.
- Hindari penggunaan placeholder; setiap visual harus memiliki makna fungsional.
- Prioritaskan kartu (card-based) sebagai unit interaksi utama.
